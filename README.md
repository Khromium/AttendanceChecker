# AttendanceChecker
点呼用ソフトだよ。プルリクを待つ戦略
バーコードリーダーで読み取った学籍番号5桁を記録します。  
10桁の場合は後ろの5桁が記録されます。  

# ライセンスについて  
- sqlite jdbc  
このソフトウェアはsqlite jdbcを使用して作られています。ライセンス及びソースコードは以下のところにあります。  
https://github.com/xerial/sqlite-jdbc  
  
  
- Apache POI  
このソフトウェアはApache POIを使用して作られています。ライセンス及びソースコードは以下のところにあります。  
http://poi.apache.org
  
- AttendanceCheckerについて  
apache2.0です。
